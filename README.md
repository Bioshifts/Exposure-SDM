# Species range shifts often speed ahead of their modeled climatic niches

This repository contains code reproduce the results from Oliveira *et al.* (2026):

1.  Download occurrence data from GBIF for species with documented range shifts in the BioShifts and CoRE databases
2.  Download environmental data from CHELSAcruts for terrestrial species and from ORAS for marine species.
3.  Fit ensamble species distribution models (SDM)
4.  Estimate modeled range shifts (SDMs)
5.  Compare modeled and documented range shifts

Please cite the original publication:

Oliveira *et al.* 2026. Species range shifts often speed ahead of their modeled climatic niches. PNAS.
